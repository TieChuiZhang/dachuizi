dealloc不被调用页面未释放

发表于 2017-11-27

问题

代理

一些自定义代理需要在viewWillDisappear方法中置空代理
- (void)viewWillDisappear:(BOOL)animated 
{
    _mapView.delegate = nil;
}
页面中有NSTimer
[NSTimer scheduledTimerWithTimeInterval:1.0 
target:self selector:@selector(updateTime:) 
userInfo:nil repeats:YES]    
也需要在viewWillDisappear中销毁NSTimer
- (void)viewWillDisappear:(BOOL)animated 
{
    [timer invalidate]; // 销毁timer
     timer = nil; // 置nil
}
viewController中block的循环引用
    由于self是__strong修饰，在 ARC 下，当编译器自动将代码中的 
    block 从栈拷贝到堆时，block 会强引用和持有self，而self
    恰好,也强引用和持有了 block，就造成了传说中的循环引用。
需要用weak修饰self
循环引用__weak typeof (self)weak_Self = self;
[_tableView mas_makeConstraints:^(MASConstraintMaker     
* make) {
        make.top.equalTo(weak_Self.view.mas_top);
        make.left.equalTo(weak_Self.view.mas_left);
          make.right.equalTo(weak_Self.view.mas_right);
          make.bottom.equalTo(weak_Self.view.mas_bottom);

    }];
